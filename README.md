# TimeCode
## About
Class providing basic functionality of encoding/decoding of time string. 
Also contains simple implementation of Berlin Clock.
## Build
There is so easy to build - use Gradle:
```batch
gradle build
```
or
```batch
gradle jar
```
## Tests
For running tests also use Gradle task:
```batch
gradle test
```
